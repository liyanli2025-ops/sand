# 聚不起的沙 · The Sand That Cannot Be Gathered

> 2026 年 3 月美伊冲突 100 天纪实 H5

一个基于 WebGL 粒子系统的纪实交互体验。用户站在伊朗"上空"，通过沉浸式星图寻访 4 个真实坐标，**长按**让粒子聚合成代表性事物，**松手**则归于沙尘。

**Live Demo**: https://liyanli2025-ops.github.io/sand/

---

## 体验流程

1. **沙之星图**：进入后看到伊朗国境轮廓由粒子勾勒，4 个城市位置有金色脉动光标
2. **AR 探索**：拖拽 / 转动手机环顾四周，WASD 或滚轮可在地图上移动
3. **瞄准进入**：屏幕中央准星对准当前可访问的坐标 → 准星亮起 → 点击屏幕进入
4. **长按聚沙**：进入场景后长按 2.5 秒，粒子聚合成事物 + 故事浮现
5. **松手散尽**：粒子崩散归于沙尘
6. **线性解锁**：完成一个坐标，下一个解锁

## 技术栈

- **Three.js 0.160**（WebGL 渲染）
- **自定义 Shader**：方向光照 + 沙粒贴图 + 涡旋聚合
- **GPU 粒子**：65,000（桌面）/ 45,000（移动端）
- **氛围层**：1,400 颗环境尘埃 + 体积光束（CSS）
- **交互**：陀螺仪（移动端）+ 鼠标拖拽 / WASD（桌面）+ 屏幕中心 raycast

## 本地运行

```bash
python -m http.server 9091

# Node
npx serve

# 然后访问 http://localhost:9091/
```
